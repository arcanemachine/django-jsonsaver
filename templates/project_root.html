What is jsonSaver?

jsonSaver is a free service that provides an easy way for users to store and retrieving arbitrary JSON data.

Use Cases

jsonSaver can be used to:

  • Save user settings and config info for frontend apps without having to setup a backend database,
  • Mock API data during testing,
  • Save random JSON snippets,
  • Do anything else you can think of!

If jsonSaver has helped make your life easier, we'd like to hear from you!

FAQ

How does jsonSaver work?

Simply register an account and confirm your email, and you can create and manage your JSON stores using either the web client or the API.

Does the data expire?

As of now, the data will not expire or be removed from the database. However, it is always a good idea to keep another backup of any important data.

How can I access the data created using this service?

  • You can lookup a public store using the form at the top of this page.
  • You can create, manage, and share JSON stores using our web-based interface.
  • You can access our API using your API key.

Where is my API key?

Your API key can be found in your user profile page.

How do I use my API key?

Simply include the following key-value pair in your HTTP header when making an API request to one of our endpoints:
Authorization: Token [your_api_key]

Example Using Javascript Fetch:

Create a new publicly-accessible JSON store:

let jsonStoreData = fetch('{{ BACKEND_SERVER_URL }}{% url 'api:jsonstore-list' %}', {
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Token e57af3bd58a56f21e9f888273546b82e7aed34cd'
  },
  body: {
    "name": "hello-world",
    "data": {
      "message": "Hello World!"
    },
    "is_public": true
  })
  .then(res => res.json())

How can I get a new API key?

You can renew your API key here.

Where is the API schema?

I haven't made one yet. The URLS used for managing your store in the browser are the same as the ones running the API.

On the web: {{ BACKEND_SERVER_URL }}{% url 'stores:jsonstore_detail_public' jsonstore_name='hello-world' %}

Using the API: {{ BACKEND_SERVER_URL }}{% url 'api:jsonstore_detail_public' jsonstore_name='hello-world' %}

How much does this cost?

This service is currently free.

Is my data private?

Any stores you create are private by default. We recommend encrypting any sensitive data before uploading it.

Can I create a public store?

Yes, you can choose to create a publicly-accessible store that can be looked up via a unique user-chosen name.

Do you sell my data?

No, that's gross.

This service currently uses no analytics whatsoever.

In the future, we may use analytical data regarding users' usage patterns to improve our service, but we will never give or sell your data or personal information to anybody.
